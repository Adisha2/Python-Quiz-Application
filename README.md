# Python Quiz Application

This is a GUI-based Python Quiz Application developed as part of an internship project.

## Features
- Tkinter-based graphical interface
- User name input
- Difficulty level selection
- JSON-based question loading
- Timer for each question
- Live score display
- Instant feedback

## Technologies Used
- Python 3
- Tkinter
- JSON
